# Adirondack Cold Water

Brand website for **Adirondack Cold Water** — small-batch, solventless **live rosin**.

> **Strong. Clean. Solventless.**

A static, single-page site. No build tools, no dependencies — just open it.

## Preview locally

```bash
# any static server works; for example:
python3 -m http.server 8000
# then visit http://localhost:8000
```

Or simply open `index.html` in a browser.

## What's inside

- `index.html` — the whole site (markup, styles, and a small vanilla-JS age gate + mobile nav)
- `assets/logo.jpg` — the brand seal
- `CLAUDE.md` — notes for AI assistants / contributors (brand tokens, conventions, compliance)

## Features

- **21+ age gate** with per-session verification
- Responsive layout (mobile nav, fluid type)
- Brand-consistent navy-on-cream design pulled from the logo
- Sections: hero, story, values, process, products/menu, contact
- Compliance disclaimer in the footer — no online sales or ordering

## Deploy

Serve the repository root as static files. GitHub Pages (from the default branch
root) works with no configuration.
